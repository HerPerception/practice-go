#  CLI Calculator

## Getting Started

 # Run the program:

##  go run cli-main.go

#  You’ll see two options:

## Calculator : Enter calculation mode.

## Help : See instructions on how to use the calculator.

## Calculator:
Type your calculation (e.g., 5+3, -2*4).
Negative numbers are supported.
The result will be shown immediately.

## Help:
Shows how to use the calculator correctly.
Use it if you’re unsure about input format.


# After Each Operation
## Continue – Perform another calculation.
## Exit – Quit the program.
